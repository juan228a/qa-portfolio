Cypress – Automatización Login & Navegación (OrangeHRM)

Proyecto de automatización realizado con Cypress sobre la demo oficial de OrangeHRM. Se automatizaron flujos de login, navegación entre módulos y validaciones básicas de interfaz. El proyecto utiliza buenas prácticas como beforeEach para preparar el estado inicial y after para finalizar sesión, garantizando independencia entre tests.

  • • Automatización de login con credenciales válidas
  • • Navegación por diferentes secciones del sistema
  • • Uso de hooks (beforeEach, after) para manejar estado
  • • Selectores CSS y selectores XPath (cypress-xpath)
  • • Tests independientes y reproducibles
  • • Preparado para escalar a Page Object Model
